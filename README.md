# Scientific-calculator-🔬 

A powerful, modern **Scientific Calculator** built with **HTML**, **CSS**, and **JavaScript** — upgraded with **Dark Mode**, **Calculation History**, **Keyboard Support**, and **Smooth Button Animations**.

This project is perfect for learning DOM manipulation, event handling, mathematical functions, UI/UX enhancements, and interactive JavaScript logic.



## 🚀 Features

### 🧮 Core Calculator Functions
- Addition, subtraction, multiplication, division  
- Parentheses support  
- Decimal support  
- Exponent operator (`^` → `**`)  

### 🔢 Scientific Functions
- `sin(x)`  
- `cos(x)`  
- `tan(x)`  
- `log(x)` (base 10)  
- `ln(x)` (natural log)  
- `sqrt(x)`  
- π and e constants  

### 🌙 Dark Mode
- One‑click theme toggle  
- Smooth transitions  
- Fully styled for both light and dark themes  

### 📜 Calculation History
- Stores previous calculations  
- Auto‑scrolling history panel  
- Newest results appear at the top  

### ⌨️ Keyboard Support
- Numbers and operators  
- Enter → Calculate  
- Backspace → Delete last character  
- Escape → Clear  
- Full typing support  

### ⚡ UI Animations
- Button hover scaling  
- Smooth theme transitions  
- Clean modern layout

## 💻 Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling, dark mode, animations  
- **JavaScript (Vanilla)** – Logic, history, keyboard support
